## How to Use This Checklist

### Overview
This checklist helps developers, organizations, and auditors verify alignment with BGI Framework principles when using BGINexus.io sustainability licenses. It provides practical verification steps for each BGI requirement.

### Checklist Structure
- **‚úÖ Full Compliance**: Requirement fully met with evidence

- **Large**: Enterprise/popular projects, >100K users, significant infrastructure

## Section 1: Biocentric Primacy (BGI ¬ß1.1)

### 1.1 Ecosystem Integrity Prioritization

**Requirement**: Systems must prioritize ecosystem integrity above human convenience

- [ ] System design documents explicitly prioritize ecosystem health
- [ ] Trade-off decisions documented with ecosystem impact as primary factor
- [ ] Examples of convenience sacrificed for ecological benefit documented

**Evidence Required**:
- Design documentation with ecosystem-first decision rationale
- Trade-off analysis showing ecosystem prioritization

**Scale**: All scales

### 1.2 Zero Net Harm Commitment

**Requirement**: Non-negotiable constraint of zero net harm to biosphere regeneration cycles

- [ ] Baseline biosphere impact assessment completed
- [ ] Quantitative "zero net harm" threshold defined for your context
- [ ] Current impact measured against zero-harm threshold
- [ ] Mitigation plan in place for impacts exceeding threshold
- [ ] High-quality carbon removal or offsets for unavoidable emissions

**Evidence Required**:
- Environmental impact assessment report
- Quantified harm metrics and thresholds
- Mitigation strategy documentation

**Scale**: Medium and Large (recommended for Small)

## Appendix C: Glossary

**Ancestral Accountability**: Traceability of decision consequences across ecological timelines (7+ generations)

**Biocentric Primacy**: Prioritization of ecosystem integrity above human convenience

**CFSA**: Coherence-First Sensitivity Analysis - Jacobian-based mathematical validation framework. *See Option B reference implementation and use cases in [`jacobian-cfsa-implementation.md`](./jacobian-cfsa-implementation.md)*

**Coherence Threshold**: Mathematical measure of system alignment stability (BGI standard: 0.7)

**Context Drift**: Deployment of system outside its original geographical/cultural context without recalibration

**Jacobian Matrix**: Matrix of partial derivatives showing sensitivity between system variables

**LOVE Protocol**: Limiting Optimization Velocity for Equilibrium - slowing optimization when coherence degrades

**Neurodivergent Cognition**: Recognition that cognitive processing varies across individuals; not treating neurotypical as default

**Sapient Partnership**: Recognition of human (especially neurodivergent) sapience with co-creation and veto authority

**Temporal Fidelity**: Synchronization with biological/seasonal rhythms rather than machine time

**Three-Tier Validation**: Biocentric impact + Sapient intent + Intergenerational consequence assessment

**Zero Net Harm**: Non-negotiable constraint of no harm to biosphere regeneration cycles
