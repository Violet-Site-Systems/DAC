About This API
This API implements the Biocentric Impact Assessment specification from the BGI Framework. It evaluates proposed actions across 6 core dimensions: Biodiversity, Water Systems, Soil Health, Air Quality, Carbon Cycle, and Regeneration Capacity. All assessments enforce the non-negotiable constraint of zero net harm to biosphere regeneration cycles.
